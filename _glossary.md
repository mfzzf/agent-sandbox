# 词汇表

## 沙箱 (Sandbox)

沙箱是一个隔离的计算环境，具备完整的操作系统能力。它是 UAgentBox 的核心运行单元，用户可以在其中安全地运行代码、安装软件、管理文件。

## 模板 (Template)

模板是沙箱的蓝图。它定义了沙箱的运行环境（如操作系统、预装软件、环境变量等）。通过构建模板，您可以快速启动具有一致环境的沙箱实例。

## API Key

用于访问 UAgentBox 服务的安全令牌。您可以在控制台获取，并通过环境变量 `AGENTBOX_API_KEY` 进行配置。

## 指标 (Metrics)

沙箱在运行过程中的资源使用情况数据，包括 CPU 使用率、内存占用等。

## 持久化集 (Persistence)

允许沙箱在重启或重新创建后保留特定目录下的文件数据。
